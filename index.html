Max/MSP

This software is created in the Max/MSP development environment. It has quite a few advanced features. It also has quite a few bugs. Please let me know if you have any feature requests, I hope to continue development on this soon.

Features

  • automatic tone row to matrix generation, and serial operations
  • set theory - with prime form, complement
  • notes displayed on a treble clef staff
  • MIDI playback of tone row and sets
  • metronome with advanced functions
  • saving! save your favorite rows and sets

Running this patch

This patch is not yet available as a standalone app. To run it, you need to get the (free) Max Runtime Environment. This was programmed in Max 6, it has not been tested in Max 7, but you are free to try. It requires a few externals, including bach.
